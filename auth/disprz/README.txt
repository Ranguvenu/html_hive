Inbiz Moodle Authentication

 How it works (user)

* Click login with Disprz Button in login page
* Authorize Moodle to get your profile info from Disprz instance
* You are now logged in


- Install the plugin

1. Add the auth plugin to /auth/Disprz 
2. In Moodle go to the plugins page for authentication and enable Disprz
4. Enter your Moodle website details, most important field: Disprz URL , SSO and other information.
